ASP.NET

¿Que es ASP .NET?

ASP.NET es un marco de desarrollo web del lado del servidor altamente desarrollado creado por Microsoft. ASP.NET se utiliza para crear páginas web dinámicas, aplicaciones web y servicios web. Después de décadas de desarrollo, el marco aún existe como ASP.NET Core[1].

ASP.NET: de Active Server Pages al framework

La funcionalidad básica de ASP se puede comparar con la de PHP o Java Server Pages (JSP). Las tres tecnologías utilizan archivos de plantilla, que contienen fragmentos de código ejecutable incrustados en contenido HTML estático. El código está rodeado de etiquetas especiales para separarlo del HTML. Los dialectos específicos de Microsoft disponibles en ese momento, VBScript, JScript y PerlScript, se utilizaron como lenguajes de programación[1].

Cuando el navegador accede a una URL ("solicitud"), el código se ejecuta en el servidor. La ejecución genera contenido HTML que se incrusta en estructuras HTML predefinidas. Por lo general, el resultado es un documento que consta de contenido HTML estático y generado dinámicamente, que se entrega en respuesta al navegador ("respuesta") y se muestra al usuario. Cualquier cambio en el contenido de la página, por ejemplo, debido a la entrada del usuario, requería un ciclo completo de solicitud-respuesta y luego una recarga de la página[2]. Aquí hay un ejemplo de código ASP:

Ejemplo

                            
                                                    <p>
                                                        The server's current time:
                                                        <%
                                                            Response.Write Now()
                                                        %>
                                                    </p>

Aplicaciones de ASP.NET

• Web Forms: permite el desarrollo rápido de aplicaciones con una rica biblioteca de controles y una experiencia de Visual Studio de arrastrar y soltar[2].

• MVC: Permite aplicar los últimos estándares web, realizar TDD y desarrollo basado en dominio (DDD) con la debida separación de responsabilidades en nuestras bibliotecas[2].

• Páginas web: combine el código del servidor con HTML para crear contenido web dinámico utilizando la sintaxis de Razor[2].

Los tres marcos ASP.NET se basan en .NET Framework y comparten la funcionalidad básica de .NET y ASP.NET. Por ejemplo, los tres marcos ofrecen un modelo de seguridad de conexión basado en enlaces y los tres comparten las mismas características de manejo de solicitudes, control de sesiones, etc., que son características principales de ASP.NET[3].

Además, los tres marcos no son totalmente independientes y la elección de uno no excluye el uso de otro. Dado que los marcos pueden coexistir en la misma aplicación web, no es raro ver componentes de aplicaciones individuales escritos usando diferentes marcos. Por ejemplo, las partes de una aplicación orientadas al cliente se pueden desarrollar en MVC para optimizar el marcado, mientras que las partes de acceso y administración de datos se desarrollan en formularios web para aprovechar controles simples y acceso a datos[3].

Carácteristicas ASP.NET

• Entorno profesional: como hemos visto, ASP.NET no es solo un marco multiplataforma, sino que ha sido compatible con Microsoft desde su inicio y se ha utilizado en entornos profesionales durante más de 25 años. Promueve buenas prácticas de monitoreo y prueba al proporcionar muchas herramientas, bibliotecas y utilidades[2].

• Documentación y comunidad: este framework debe destacarse por su abundante documentación, generada por la numerosa y muy activa comunidad de desarrolladores, comunidad que además participa en la mejora continua. Todo el código de .NET, incluido ASP.NET, está en GitHub y .NET tiene más de 100 000 contribuciones, de hecho, 3700 empresas han contribuido[2].

• Rendimiento: Otro punto a destacar son las enormes mejoras de rendimiento que supuso la reescritura del Framework prácticamente desde cero, el rendimiento evoluciona con cada versión y ya nos permite utilizar ASP.NET para desarrollos que requieren gran precisión e inmediatez, como Ejemplos de Aplicaciones. de aprendizaje automático. Es muy interesante ver las comparaciones de TechEmpower Round 20 de donde se toman estos datos[2].

• Versatilidadcomo se mencionó varias veces en este artículo, ASP.NET es multiplataforma, es decir, ya no depende de los sistemas Windows, sino que puede distribuirse en múltiples entornos, incluidos Linux, MacOS, contenedores Docker e incluso en la nube, lo que brinda una versatilidad de la que pocas estructuras pueden estar orgullosas. Además, se integra con interfaces populares como React, Vue y Angular[2].

Razor Pages: evolución de ASP.NET Web Pages

Razor Pages llena un nicho similar al que ocupaban las páginas web ASP.NET en el pasado. Se utiliza mejor cuando el patrón modelo-vista-controlador genera una sobrecarga innecesaria. Si, por ejemplo, desea crear una página de destino, con Razor Pages esto se puede implementar fácilmente. La sintaxis de Razor se usa para crear la plantilla de página. Como es habitual en el universo .NET, se pueden utilizar C# y Visual Basic como lenguajes de programación[1]. Tenga en cuenta que @ se usa como un signo de apertura en lugar de las etiquetas de código de apertura y cierre habituales en los lenguajes de modelado:

Ejemplo

                            
                                                @page
                                                @model HelloWorldModel
                                                @if (Name != null) {
                                                    <p>Hello dear @Name</p>
                                                }
                                                <form method="post">
                                                    <p>
                                                        <label asp-for="Name"></label>
                                                        <input class="form-control" asp-for="Name" />        
                                                    </p>
                                                    <input type="submit" value="Say Hello" />
                                                </form>
