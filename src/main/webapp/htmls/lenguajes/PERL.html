PERL

¿Qué es PEARL?

Perl es un lenguaje de programación genérico desarrollado originalmente para la manipulación de texto y ahora se usa para una amplia gama de tareas que incluyen administración de sistemas, desarrollo web, programación de redes, desarrollo de GUI, etc[1].

El lenguaje pretende ser práctico (fácil de usar, eficiente, completo) en lugar de lindo (pequeño, elegante, mínimo). Sus características principales son que es fácil de usar, admite programación de procedimientos y orientada a objetos (OO), tiene un potente soporte incorporado para procesamiento de textos y tiene una de las colecciones de módulos de terceros más impresionantes del mundo[1].

Varias definiciones de Perl se dan en perl, perlfaq1 y probablemente en otros lugares. A partir de esto podemos determinar que Perl es un poco diferente de una persona a otra, pero muchas personas encuentran que vale la pena escribir sobre él[1].

¿Qué es POD?

Pod es un lenguaje de marcado fácil de usar que se utiliza para escribir documentación para Perl, programas Perl y módulos Perl. Hay varios traductores disponibles para convertir Pod a diferentes formatos como texto sin formato, HTML, páginas man, etc[2]. El marcado de pod consta de tres tipos de párrafo básicos:

• Párrafo común: puede usar códigos de formato en párrafos normales, en negrita, en cursiva, estilo de código, hipervínculos y más[2].

• Párrafo literal: los párrafos literales generalmente se usan para presentar un bloque de código u otro texto que no requiere un análisis o formato especial y no debe romperse[2].

• Párrafo de comando: un párrafo de comando se usa para el manejo especial de fragmentos completos de texto, generalmente como encabezados o partes de listas. Todos los párrafos de comando comienzan con =, seguido de un identificador, seguido de texto arbitrario que el comando puede usar como mejor le parezca[2].

Los comandos actualmente reconocidos son:

                            =pod
                            =head1 Heading Text
                            =head2 Heading Text
                            =head3 Heading Text
                            =head4 Heading Text
                            =over indentlevel
                            =item stuff
                            =back
                            =begin format
                            =end format
                            =for format text...
                            =encoding type
                            =cut
