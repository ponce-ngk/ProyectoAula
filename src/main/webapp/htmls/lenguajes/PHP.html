PHP

¿Que es PHP?

PHP (acrónimo recursivo de PHP: preprocesador de hipertexto) es un lenguaje de código abierto muy extendido, especialmente adecuado para el desarrollo web que se puede incrustar en HTML. En lugar de usar muchos comandos para mostrar HTML (como en C o Perl), las páginas PHP contienen HTML con código que hace "algo" (en este caso dice "¡Hola, soy un script PHP!). ¿El código PHP está encerrado en etiquetas especiales de inicio y fin que le permiten insertar y salir de "PHP modo"[1].

Lo que distingue a PHP de algo del lado del cliente como JavaScript es que el código se ejecuta en el servidor, genera HTML y lo envía al cliente. El cliente recibirá el resultado de ejecutar el script, aunque no sabe cuál era el código subyacente. El servidor web también se puede configurar para procesar todos los archivos HTML con PHP, por lo que no hay forma de que los usuarios sepan lo que tienen bajo la manga[2].

Lo mejor de usar PHP es su extrema simplicidad para principiantes, pero al mismo tiempo ofrece muchas funciones avanzadas para programadores profesionales. No tenga miedo de leer la larga lista de características de PHP. En unas horas puedes empezar a escribir tus primeros guiones[1].

Ejemplo

                            
                                <DOCTYPE html>
                                <html lang=”es”>
                                <head>
                                    <title>Título del documento </title>
                                <head>
                                <body>
                                    < ?php
                                        echo "¡Hola, soy un script de PHP!";
                                    ?>
                                <body>
                                </html>

¿Qué puede hacer PHP?

Script del lado del servidor. Esta es la zona más tradicional y el foco principal. Se necesitan tres cosas para que funcione: el analizador PHP (módulo o servidor CGI), un servidor web y un navegador web. Debe ejecutar el servidor con una instalación PHP conectada. Se puede acceder a la salida del programa PHP con un navegador al ver la página PHP a través del servidor. Todos estos se pueden ejecutar en su máquina si tiene experiencia con la programación PHP. Para obtener más información, consulte la sección Instrucciones de instalación[2].

Guión de línea de comandos. Se puede crear y ejecutar un script PHP sin necesidad de un servidor o navegador. Solo necesita el analizador PHP para usarlo así. Este tipo de uso es ideal para scripts que se ejecutan regularmente usando cron (en * nix o Linux) o el planificador (en Windows). Estos scripts también se pueden usar para tareas simples de procesamiento de texto. Para obtener más información, consulte Uso de PHP en la línea de comandos[2].

Escribir aplicaciones de escritorio. PHP probablemente no sea el lenguaje más adecuado para crear aplicaciones de escritorio con una interfaz gráfica de usuario, pero si está familiarizado con PHP y desea utilizar algunas características avanzadas de PHP en aplicaciones del lado del cliente, puede utilizar PHP-GTK para escribir dichos programas. De esta forma también es posible escribir aplicaciones independientes de la plataforma. PHP-GTK es una extensión de PHP, no disponible en la distribución principal. Si estás interesado en PHP-GTK, puedes visitar su sitio web[2].

Carácteristicas

• Fácil de usar: PHP 5 es un lenguaje muy fácil de aprender en comparación con otros lenguajes utilizados para el mismo propósito como JAVA o ASP. Por ello, no es necesario realizar un estudio muy concienzudo de sus funciones para poder realizar sencillos programas que solucionen la mayoría de nuestros problemas diarios[3].

• Embebido en HTML: Las páginas escritas en PHP son páginas HTML simples que contienen, además de las etiquetas normales, el programa que desea ejecutar[3].

• Multiplataforma: PHP 5 se ejecuta en muchas plataformas, sistemas operativos y servidores existentes. Es compatible con los tres principales servidores del mercado: Apache, Microsoft Internet Information Server y Netscape Enterprise Server[3].

• Licencia Open Source: la licencia de código abierto significa que podemos descargar e inspeccionar el código fuente de PHP 5 de forma gratuita. La principal consecuencia es que el coste del producto en la mayoría de los casos es de 0 euros. Tener el código fuente de PHP 5 es útil, entre otras cosas, para poder construir nuestro servidor personalizado[3].

• Multitud de Extensiones: Database Access es una herramienta poderosa que implementa soporte nativo para 15 sistemas populares de administración de bases de datos. Por el lado del protocolo, puede contar con extensiones que controlan el acceso a LDAP, IMAP o POP3. También se ha agregado soporte para crear imágenes en tiempo de ejecución, gracias a la biblioteca GD[3].

• Velocidad e incorporación de objetos: El nuevo motor Zend 2.0 acelera los procesos de ejecución de código. Además, incorpora un nuevo modelo de objetos que le permite crear clases y métodos privados, protegidos y públicos, clases abstractas e interfaces[3].

• Gran comunidad de apoyo: PHP 5 fue escrito bajo los auspicios de Open Source. Por lo tanto, existe una comunidad que apoya su desarrollo colaborativo. La principal ventaja es que existen muchas páginas, listas de correo y foros de discusión cuyo tema de conversación es el uso de este lenguaje de programación[3].

Operadores de concatenación

Ambas variables se escriben con el resto del texto que se va a visualizar en el string señalado con las comillas dobles. PHP reconoce automáticamente las variables mediante el símbolo del dólar ($) y coloca tras ellos los valores correspondientes.

                            <?php
                                $author1 = "John Doe";
                                $author2 = "Max Mustermann";
                                echo "<h1>Hello World!</h1>
                                <p>This dynamic web page was created by $author1 and $author2.<p>";
                            ?>

Realización de cálculos con variables

Ejemplo

                            <?php
                                $numero1 = 10;
                                $numero2 = 5;
                                $adicion = $numero1 + $numero2; 
                                $sustraccion = $numero1 - $numero2; 
                                $multiplicacion = $numero1 * $numero2; 
                                $division = $numero1 / $numero2; 
                                $exponenciacion = $numero1 ** $numero2; 
                                ?>
                                <?php
                                echo "Resultado de la adición: " . $adicion ."<br />"; 
                                echo "Resultado de la sustracción: " . $sustraccion . "<br />"; 
                                echo "Resultado de la multiplicación: " . $multiplicacion . "<br />";
                                echo "Resultado de la división: " . $division . "<br />";
                                echo "10 elevado a la quinta potencia (10^5): " . $exponenciacion . "<br />";
                                echo "Raíz de 81: " . sqrt(81) . "<br />";
                            ?>

Las superglobals $_GET y $_POST

Las superglobals$_GET y $_POST contienen una serie de variables en forma de llaves que permiten llegar a los valores vinculados a dichas llaves. Hablaremos en detalle de este tema cuando nos centremos en las superglobals $_GET y $_POST .[4]

Una característica importante de los lenguajes de programación es que brindan la capacidad de analizar la entrada del usuario y pasar valores a otros scripts. Para hacer esto, PHP se basa en las superglobales $_GET y $_POST, variables de sistema predefinidas disponibles en todos los dominios de validez. Al igual que las matrices asociativas (campos de datos), $_GET y $_POST almacenan un conjunto de variables como cadenas en una variable[4].
