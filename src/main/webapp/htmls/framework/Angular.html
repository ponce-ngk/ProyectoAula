Historia

El framework Angular ha sido completamente rediseñado en la versión 2.0, pasando de AngularJS a Angular. Para no abandonar a los desarrolladores de AngularJS, Google continúa apoyando y manteniendo AngularJS. Actualmente, AngularJS está en la versión 1.6. y Angular está en la versión 4.0 [1].

El concepto de Aplicaciones de una sola página (SPA) comenzó a discutirse en 2003, pero no fue hasta 2005 que se utilizó por primera vez el término. Como podemos ver, es un tipo de aplicación relativamente nuevo, sin embargo, su uso se está extendiendo rápidamente y hoy hemos descubierto su presencia en aplicaciones conocidas como Gmail, Outlook, Facebook y Twitter [1].

El auge de este tipo de aplicaciones se debe en parte a las ventajas que ofrecen frente a otras soluciones, pero sobre todo a la enorme inversión que están haciendo las grandes empresas tecnológicas para desarrollar los frameworks que permitan su desarrollo. Este es el caso de Google con el framework Angular, o Facebook con el framework React.js [1].

AngularJS fue desarrollado en 2009 por Miško Hevery y Adams Abrons. Originalmente era el software detrás de un servicio de almacenamiento de archivos JSON en línea, pero más tarde, el proyecto se abandonó y AngularJS se lanzó como una biblioteca de código abierto. Adams Abrons abandonó el proyecto, pero Miško Hevery, como empleado de Google, continúa desarrollando y manteniendo el framework [2].

Angular JS 1.X

Fue lanzado inicialmente en octubre de 2010 (hace 7 años).

Su objetivo es simplificar el desarrollo y las pruebas de aplicaciones al proporcionar un marco para las arquitecturas Model View Control (MVC) y Model View Model (MVVM) para los clientes, así como los componentes que se usan comúnmente en las aplicaciones [2].

Angular 2

Fue lanzado en septiembre de 2016.

Angular 1.x no está diseñado teniendo en cuenta la compatibilidad móvil, mientras que Angular 2 está orientado a dispositivos móviles [3].
